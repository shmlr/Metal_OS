# Metal OS / Welding OS 2.0-dev — Phase 1

Это первая безопасная модульная миграция Metal OS.

## Что сделано

- Исходный рабочий `index.html` разделён на:
  - `index.html`
  - `css/app.css`
  - `js/app.js`
  - `js/pwa.js`
- Логотип и PWA-иконки лежат в `assets/`.
- Добавлены `manifest.webmanifest` и `service-worker.js`.
- Бизнес-логика специально не переписывалась, чтобы сохранить поведение текущей версии.

## Проверка

1. Открыть `index.html`.
2. Проверить логотип в шапке.
3. Проверить ссылку Telegram внизу.
4. Нажать «Рассчитать».
5. Проверить КП.
6. Проверить на телефоне.
