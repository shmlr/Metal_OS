# Metal OS 2.0 — конструктивные схемы

Расчёт начинается с выбора реальной конструкции: "Стена + 1 стойка", "Стена + 2 стойки", "4 стойки", "Между двумя стенами", "Консольный".

В этой сборке добавлены:
- `schemes/canopy/*.json`
- `js/scheme-constructor.js`
- блок выбора конструкции в интерфейсе

Текущий режим — совместимый слой поверх старого расчётного движка.
Следующий этап — вынести расчёт навесов в отдельный движок, который напрямую читает JSON-схему.
